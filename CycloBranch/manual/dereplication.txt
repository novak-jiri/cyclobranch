\section msdatadereplication Dereplication in MS Data

<p>
CycloBranch supports the dereplication of compounds in single mass spectra (MALDI-MS). The mode "Compare Peaklist(s) with Database - MS or MSI data" must be selected (see [Mode](@ref settings_mode)).
The detectable ion types are listed in [Ion Types in Theoretical Spectra](@ref settings_iontypes). An output report example is shown in the following picture.
</p>

<p>
\image html msdataoutputreport.png "Single mass spectrum was searched against an in-house database of siderophores - output report."
\image latex msdataoutputreport.png "Single mass spectrum was searched against an in-house database of siderophores - output report." width=16cm
</p>

<p>
The detail window is opened by a double-click on a row in the output report if a linear peptide was identified. The Enter key can also be used if the row is selected.
See also [Linear Peptide Detail Window](@ref lineardetailwindow) and [Toolbar for Linear Peptides](@ref lineartoolbar). The differences for MS data are following:
<ul>
<li> Only the windows on the left side are available.
<li> The red peaks in the graphical visualization of a mass spectrum were assigned to compounds in a database.
</ul>
</p>

<p>
The following details are listed in the table of peaks:
<ul>
<li> Ion Type - a name of a theoretical peak including information about its charge (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is disabled)
<li> Pattern Type - a name of a theoretical isotope pattern including information about its charge (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> Theoretical m/z - a theoretical mass-to-charge ratio of the peak
<li> Theoretical intensity [%] - theoretical intensity of the peak (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> Experimental m/z - a mass-to-charge ratio of a matched peak
<li> Relative Intensity [%] - a relative intensity of a matched peak
<li> Absolute Intensity - an absolute intensity of a matched peak
<li> Error [ppm] - a ppm error
<li> Score - the angle distance calculated between theoretical and experimental isotopic patterns (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> FDR - false discovery rate of a matched isotopic pattern (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled) 
<li> Summary Formula - a summary formula of a neutral molecule
<li> Name - a name of a compound
<li> Reference - a link into a database if available (ChemSpider, PubChem or Norine)
</ul>
</p>

<p>
Note that theoretical peaks which were not assigned to experimental peaks are not reported in the table.
</p>

<p>
\image html msdatadetailwindow.png "Single mass spectrum was searched against the in-house database of siderophores - detail window."
\image latex msdatadetailwindow.png "Single mass spectrum was searched against the in-house database of siderophores - detail window." width=16cm
</p>

\section lcmsdatadereplication Dereplication in LC-MS data

<p>
CycloBranch supports also the dereplication of compounds in LC-MS data. The mode "Compare Peaklist(s) with Database - MS or MSI data" must be selected (see [Mode](@ref settings_mode)).
The data files containing spectra from multiple LC-MS scans are processed automatically. In this case, each row in the output report corresponds to one scan.
</p>

<p>
\image html lcmsdataoutputreport.png "Batch processed LC-MS data - output report."
\image latex lcmsdataoutputreport.png "Batch processed LC-MS data - output report." width=16cm
</p>

<p>The compounds detected in the scans are summarized in the [Summary Table of Matched Peaks](@ref summarypeakstable). The "ID" in the summary table of matched peaks refers to the "Spectrum ID"
in the output report, and it is the scan number. See the [Summary Table of Matched Peaks](@ref summarypeakstable) in the following image and the corresponding chromatogram (see also [Chromatogram](@ref chromatogram)). 
</p>

<p>
\image html batch-lc-ms-1-ferri-ferrioxamineB.png "List of peaks assigned to ferri-ferrioxamine B in the batch-processed LC-MS run."
\image latex batch-lc-ms-1-ferri-ferrioxamineB.png "List of peaks assigned to ferri-ferrioxamine B in the batch-processed LC-MS run." width=16cm
</p>

<br/>

<p>
\image html eic.png "Extracted ion chromatogram (EIC) of ferri-ferrioxamine B."
\image latex eic.png "Extracted ion chromatogram (EIC) of ferri-ferrioxamine B." width=16cm
</p>

\section msidatadereplication Dereplication in MSI Data

<p>CycloBranch supports the dereplication of compounds in imaging mass spectra (MSI). The mode "Compare Peaklist(s) with Database - MS or MSI data" must be selected (see [Mode](@ref settings_mode)).
An imzML file must be used as the [input file](@ref settings_peaklistfile) in the settings dialog. A row in the output report corresponds to single mass spectrum with the coordinates [X,Y].
</p>

<p>
\image html msidataoutputreport.png "A mass spectrometry imaging dataset was searched against the in-house database of siderophores - output report."
\image latex msidataoutputreport.png "A mass spectrometry imaging dataset was searched against the in-house database of siderophores - output report." width=16cm
</p>

<p>If MSI data are analyzed the [Summary Table of Matched Peaks](@ref summarypeakstable) includes compounds detected in all single mass spectra. 
The [CrossVis](@ref crossvis) tool visualizes the localization of compounds in an optical image. 
<p> 

<p>
\image html msi-summarytable.png "Summary table of matched peaks - desferri-ferrioxamine B was filtered by its theoretical m/z ratio."
\image latex msi-summarytable.png "Summary table of matched peaks - desferri-ferrioxamine B was filtered by its theoretical m/z ratio." width=16cm
</p>

<br/>

<p>
\image html desferri-ferrioxamineB-msi.png "CrossVis - detail of a region containing desferri-ferrioxamine B."
\image latex desferri-ferrioxamineB-msi.png "CrossVis - detail of a region containing desferri-ferrioxamine B." width=16cm
</p>

\section dereplicationdatabases Databases of Compounds

<p>The databases of sequences/compounds can be created/edited using [Sequence/Compound Database Editor](@ref sequenceeditor). The same database can be used in both MS and MS/MS mode. However, the column "Sequence"
must be defined in MS/MS mode. See [List of Available Sequence/Compound Databases](@ref availablesequencedatabases).</p>
