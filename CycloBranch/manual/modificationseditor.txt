\subsection subsec_modificationseditorcolumns Columns

<p>
<ul>
<li> Name - Name of an N-terminal or a C-terminal modification.
<li> Summary Formula - Summary formula of a modification (e.g., H2C2O for acetylation or HNO-1 for amidation). Negative numbers of atoms are supported (e.g., H-2O3N-5 means 
remove two hydrogens, add three oxygens and remove five nitrogens).
<li> Monoisotopic Mass - Monoisotopic mass corresponding to a modification. The value is recalculated when the summary formula is changed.
<li> N-terminal - When checked, the modification is considered to be N-terminal.
<li> C-terminal - When checked, the modification is considered to be C-terminal.
</ul>
</p>

<hr>

\section modificationseditorformat Format of Modification Databases

<p>The list of modifications is a text file (*.txt) containing one modification per line. Each line contains the following items separated by tabulars: 
<ul>
<li> name of a modification
<li> summary formula of a modification (important: an exact mass of the block is calculated from this value)
<li> monoisotopic mass (the value is ignored; the mass is automatically calculated from the summary formula)
<li> the value "1", when the modification is N-terminal; "0" otherwise
<li> the value "1", when the modification is C-terminal; "0" otherwise
</ul>

A line starting with the hash '#' is a comment which is not processed.
</p>
