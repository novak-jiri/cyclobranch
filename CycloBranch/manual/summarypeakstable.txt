\section summarypeakstableinterface Summary Table of Matched Peaks

<p>The table is opened using the command "Tools -> Summary Table of Matched Peaks" from the [Main Window](@ref mainwindow).
Use F1 to open this help.
</p>

<p>The table is refreshed automatically when the content of the [Output Report](@ref outputreport) in the main window is changed. The content of the table is
generated from the rows in the [Output Report](@ref outputreport) which are currently visible, i.e., you can use the 'Filter' button in the main window
to limit the rows from which the summary table of peaks is generated.</p>

\subsection subsec_summarypeakstablefilter Text Filter

<p>See details [here](@ref filter). The text filter contains a suggester of names of identified compounds if the mode "Compare Peaklist(s) with Database - MS, LC-MS, MSI" (see [Mode](@ref settings_mode)) was used.
The molecular formulas are used instead of the names if the mode "Compound Search - MS, LC-MS, MSI" was used. The arrows \< and \> can be used to browse all identified compounds.
A compound can also be directly seen in [Chromatogram](@ref chromatogram) window (LC-MS) or [CrossVis](@ref crossvis) tool (MSI data).</p>

\subsection subsec_summarypeakstablecolumns Columns

<p>
The following columns are reported for MS, LC-MS and MSI data:
<ul>
<li> ID - Spectrum ID in the [Output Report](@ref outputreport)
<li> Ion Type - a name of a theoretical peak including information about its charge (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is disabled)
<li> Pattern Type - a name of a theoretical isotope pattern including information about its charge (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> Theoretical m/z - a theoretical mass-to-charge ratio of the peak
<li> Theoretical intensity [%] - theoretical intensity of the peak (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> Experimental m/z -  a mass-to-charge ratio of a matched peak
<li> Relative Intensity [%] - a relative intensity of a matched peak
<li> Absolute Intensity - an absolute intensity of a matched peak
<li> Error [ppm] - a ppm error 
<li> Score - the angle distance calculated between theoretical and experimental isotopic patterns (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> FDR - false discovery rate of a matched isotopic pattern (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled) 
<li> Summary Formula - a summary formula of a neutral molecule
<li> Name - a name of a compound
<li> Reference - a link into a database if available (ChemSpider, PubChem or Norine)
</ul>
</p>

<p>
The following column is reported for LC-MS data:
<ul>
<li> Time - retention time (time units are dataset dependent)
</ul>
</p>

<p>
The following columns are reported for MSI data:
<ul>
<li> Coordinate X - x coordinate of a single mass spectrum
<li> Coordinate Y - y coordinate of a single mass spectrum
</ul>
</p>

<p>
The following columns are reported for MS/MS data:
<ul>
<li> ID - Result ID in the [Output Report](@ref outputreport)
<li> Group ID - identifier of a group of peptide sequence candidates generated from the same path in a de novo graph (only if mode is set up to "De Novo Search Engine"; see [Mode](@ref settings_mode))
<li> Fragment Type - a name of a theoretical peak including information about its charge
<li> Theoretical m/z - a theoretical mass-to-charge ratio of the peak
<li> Theoretical intensity [%] - theoretical intensity of the peak (if [Generate Full Isotope Patterns](@ref settings_isotopepatterns) is enabled)
<li> Experimental m/z -  a mass-to-charge ratio of a matched peak
<li> Relative Intensity [%] - a relative intensity of a matched peak
<li> Absolute Intensity - an absolute intensity of a matched peak
<li> Error [ppm] - a ppm error
<li> Summary Formula - a summary formula of a fragment ion
<li> Sequence - a sequence of building blocks corresponding to the peak
</ul>
</p>
