<p>The tool is opened using the command "Tools -> Chromatogram" from the [Main Window](@ref mainwindow).

Use F1 to open this help.

<p><b>Mouse Spectrum ID Selection Tool (CTRL + T)</b>
\htmlonly
<img src="64.png">
\endhtmlonly
\latexonly
\includegraphics{64.png}
\endlatexonly
</p>
<p>If the button is enabled, the mouse can be used to select a range of IDs to be shown.
<ul>
<li> Left mouse button - press and hold the button to draw a rectangle determining the minimum and maximum values of IDs; release the button to apply the changes.
<li> Right mouse button - press the button during the drawing of a rectangle to cancel the drawing.
<li> Middle mouse button - reset the range of IDs.
</ul>
</p>

<p>
If the button is disabled, the mouse can be used to shift the interval of IDs to the left or right. Press and hold the left mouse button to shift the chromatogram.
</p>

<p>
<b>Spectrum ID</b>
<ul>
<li> First field - a minimum value of ID to be shown.
<li> Second field - a maximum value of ID to be shown.
<li> "Set" button (or Enter key) - apply the changes.
<li> "Reset" button - reset the range of IDs.
</ul>
</p>
