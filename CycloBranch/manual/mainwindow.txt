<p><b>Export to HTML (CTRL + Shift + H)</b>
\htmlonly
<img src="77.png">
\endhtmlonly
\latexonly
\includegraphics{77.png}
\endlatexonly
</p>
<p>An output report is exported into a HTML page.</p>

<p>
\image html htmlexport.png "HTML Export Dialog."
\image latex htmlexport.png "HTML Export Dialog." width=10cm
</p>

<p>
Summary:
<ul>
<li> Parameters - export the search parameters
<li> De Novo Graph - export the de novo graph (if "De Novo Search Engine" [Mode](@ref settings_mode) was used)
<li> Log Window - export the content of Log Window
<li> Output Report Table - export the [Output Report](@ref outputreport) in the main window
<li> Summary Table of Matched Peaks - export the summary table of matched peaks; see [Summary Table of Matched Peaks](@ref summarypeakstable)
</ul>
</p> 

<p>
Individual Rows in Output Report (the following data are exported for each row in the table):
<ul>
<li> Peaks Table - export a table of peaks
<li> Unmatched Experimental Peaks - export unmatched experimental peaks
<li> Unmatched Theoretical Peaks - export unmatched theoretical peaks (MS/MS data only)
<li> Details of Identification - export identification details (MS/MS data only)
</ul>
</p>

<p>
The tables in the output report can be sorted using any column. This feature uses <a href="http://www.jquery.com/">jQuery</a> and requires an on-line connection. 
If the feature does not work, the following paths in a source code of the generated HTML report must be fixed:
<pre>
https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js
https://cdnjs.cloudflare.com/ajax/libs/jquery.tablesorter/2.23.3/js/jquery.tablesorter.min.js
https://cdnjs.cloudflare.com/ajax/libs/jquery.tablesorter/2.23.3/css/theme.green.min.css
</pre>
</p>
